# ThemeManager
首页感谢Draveness大神的开源！！本demo基于 [DKNightVersion](https://github.com/Draveness/DKNightVersion)  的一键换主题，只用于学习。

以下改变
1.原有一次加载所有主题的机制，改为每次切换之后去加载。
2.原有TEXT文件存储方式，改为pilst文件（可以先预先配置，也可以网络下载到本地）
3.新增字体的切换，和UITabBarItem 支持
